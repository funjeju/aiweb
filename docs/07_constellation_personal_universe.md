# 개인 웹 = "내 우주" 별자리 컨셉 (개인 홈페이지 방향 전환)

기존 포트폴리오형 개인 빌더를 넘어, 개인 웹사이트를 **"내 우주를 키우고 꾸미는"** 감성 공간으로 재정의한다. 핵심은 "페이지를 만든다"가 아니라 "내 별자리 우주를 가꾼다"는 경험.

## 1. 별자리 생성

- 유저의 **이름 + 좋아하는 색 + 좋아하는 숫자**를 시드로 결정론적 난수(PRNG) 생성 → 유저마다 고유하고 항상 동일한 별자리.
- 별자리는 **실제 항성 카탈로그**(HYG / Yale Bright Star, 공개 데이터)의 실제 별들을 조합해 구성.
- 유저 지역 하늘 기반: 위도·경도(+시각)로 그 지역에서 보이는 실제 밤하늘 위에 유저만의 별자리가 중앙에서 빛난다. (정밀 천구 계산은 astronomy-engine 등 필요 — 단계적; MVP는 위도 근사)

## 2. 메인 화면 = 별자리 우주

- 배경: 무수한 실제 별들의 밤하늘.
- 중앙: "○○○님의 별자리"가 빛남.
- 별자리를 중심으로 **희미한 연결선**이 전후좌우로 이어지며 메뉴(별자리 노드)로 연결.
- 기본 메뉴: 내 소개, 다이어리(AI와 대화하며 정리·기록), 나만의 갤러리.
- 어드민에서 **모듈(메뉴)을 추가** — 시스템에 추가 가능한 모듈 카탈로그가 어드민에 존재해야 함.

## 3. 랜덤 이웃 구경

- "랜덤 이웃 구경" 버튼 → 유저 별자리 중심으로 다른 유저들의 별자리가 펼쳐짐.
- 다른 유저 별자리 클릭 → 그 사람의 우주를 관찰/방문.

## 4. 별자리 꾸미기 시스템 ("내 우주를 꾸민다")

- **별 꾸미기**: 색상/반짝임/크기/오라/꼬리별. 각 별에 이름·의미 부여(인생 사건·추억·목표 연결).
- **연결선 꾸미기**: 손그림/점선/오로라/빛 흐름/먼지 입자 등.
- **배경 하늘 테마**: 보랏빛 은하/빈티지 천문도/노을→밤/안개 밤/제주 감성/픽셀 우주.
- **떠다니는 오브젝트**: 행성·유성·달·우주선·동물별자리·폴라로이드·메모·파티클 (다꾸 스티커의 우주 버전).
- **메뉴 아이콘 우주화**: 내소개=행성, 다이어리=달 일기장, 갤러리=사진 성운, 링크=워프 게이트.
- **AI 감정 기반 변화**: AI 다이어리 연동 — 행복한 날 밝은 별↑, 우울한 날 푸른 안개, 목표 달성 시 새 별 생성. 감정 기록이 시각적으로 누적.
- **사운드/분위기**: 밤바람·파도·LP 노이즈·피아노·우주 환경음, 별 클릭 효과음.
- **희귀 아이템/수집**: 계절 이벤트·방문기록·AI 활동량에 따라 희귀 별/한정 스킨/특별 유성/감정 오브젝트 획득 → 장기 방문 동기.

## 핵심 방향성

홈페이지 제작 툴처럼 느껴지면 안 된다. "페이지를 만든다"가 아니라 "내 우주를 키우고 꾸민다". 감성·정체성·탐험·수집이 결합된 개인형 디지털 공간.

## MVP 단계 (검토 합의)

- **1단계 (핵심 경험)**: 시드 별자리 생성 + 실제 별 배경(Canvas) + 중앙 별자리 + 메뉴 노드(내소개/다이어리/갤러리) + 연결선 + 클릭 진입.
- **2단계 (소셜+꾸미기)**: 랜덤 이웃 구경 + 기본 꾸미기(별/연결선/배경 테마) + 어드민 모듈 추가 시스템.
- **3단계 (감성 심화)**: AI 다이어리 감정 변화 + 사운드 + 떠다니는 오브젝트 + 수집/희귀템.

## 기술 메모

- 별 카탈로그: HYG database(공개 JSON, RA/Dec/등급). 밝은 별만 필터링해 성능 확보.
- 렌더링: Canvas 2D(또는 추후 WebGL/three.js). 모바일 성능 고려해 별 수 제한.
- 시드 PRNG: mulberry32 등 경량 결정론적 생성기.
- 지역 천구 정밀: astronomy-engine(무겁다) — 2단계 이후. MVP는 위도 기반 근사.
- 데이터: 기존 personals와 별개 축(별자리 모드). PersonalSchema 확장 or 신규 universe 컬렉션 검토.
